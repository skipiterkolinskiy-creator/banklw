# Z-Банк для LimeWorld

Telegram-бот игровой экономики LimeWorld. Внутри: регистрация, карты, Z-ID для анонимных переводов, криптокошелек, кредиты до 5 000 000, минусовой баланс при просрочке, банкротство, казна, мэр, налоги, казино и админ-панель.

## Первый запуск игрока

При `/start` бот регистрирует клиента и пишет:

```text
Привет! Я Олег! Твой виртуальный помощник!

Баланс: 35 000 лаймов
История:
1.
2.
3.
4.
5.

Ежедневный бонус нет.
```

Игрок сразу получает карту, Z-ID и криптокошелек.

## Игроки

- `/start` - регистрация и главный экран.
- `/balance` - баланс, карта, Z-ID, вклад, кредит, криптокошелек.
- `/pay @user 100` - перевод по нику или ответом на сообщение.
- `/zpay Z1234567890 100` - анонимный перевод по Z-ID.
- `/crypto` - данные криптокошелька.
- `/buycrypto 10` - купить LWC.
- `/sellcrypto 10` - продать LWC.
- `/cryptopay LWABC... 10` - перевод крипты.
- `/deposit 1000` и `/withdraw 1000` - вклад.
- `/loan 5000000` - кредит. Если просрочить, долг уходит в минус баланса.
- `/repay 1000` - погашение кредита.
- `/bankrupt` - самобанкротство при долге или минусе: долг списывается, счет временно замораживается.
- `/history` - история операций.
- `/top` - топ по балансу.

## Казна, мэр и налоги

- `/treasury` - состояние казны.
- `/donate 1000` - донат в казну, можно использовать для штрафов и коммуналки.
- `/tax` - посмотреть свой налоговый счет.
- `/paytax` или `/paytax 2000` - оплатить налог.
- `/mayorwithdraw 1000` - мэр снимает деньги из казны.
- `/settax 2000` - мэр или админ меняет недельный налог.
- `/billtax` - админ выставляет налог всем активным клиентам.

Если игрок долго не платит налог, счет замораживается.
Без отдельного планировщика налог запускается вручную командой `/billtax`.

## Казино

- `/casino` - подсказка по играм.
- `/number 100 7` - угадай число от 1 до 10.
- `/dice 100` - кубик, выигрыш на 5-6.
- `/blackjack 100` - 21 очко.

При проигрыше в истории пишется `ООО "Бнал"`, при выигрыше - `ООО "Бул казик"`.

## Уведомления

При переводе получателю приходит личное уведомление:

```text
Новое уведомление!
900:
Вы получили зачисление от Z-ID ****1234 / КриптоКошелек LWABCDEF / карта ****1234
Сумма: 100 лаймов
Ваш баланс: ...
```

## Админ-панель

`/panel` работает только в чате из `ADMIN_CHAT_ID`. Если `ADMIN_USER_IDS` пустой, команды доступны всем, кто пишет в этом спец-чате. Если заполнен, доступны только указанным Telegram user ID.

Админ-команды:

- `/check @user` - проверка аккаунта.
- `/freeze @user причина` или `/banbank @user причина` - блокировка/заморозка аккаунта.
- `/unfreeze @user` или `/unbanbank @user` - разморозка.
- `/verify @user` - отметить аккаунт проверенным.
- `/give @user 1000 причина` - выдать деньги.
- `/take @user 500 причина` - списать деньги, может увести баланс в минус.
- `/setbalance @user 10000 причина` - установить баланс.
- `/setmayor @user` - назначить мэра.
- `/billtax` - выставить налог всем активным клиентам.

Команды можно писать ответом на сообщение игрока: `/freeze причина`, `/give 1000 награда`.

## Анти-альт защита

Если несколько новых аккаунтов быстро переводят крупные стартовые суммы одному игроку, бот автоматически замораживает получателя и пишет в админ-чат. Настройки:

- `ALT_NEW_ACCOUNT_DAYS`
- `ALT_TRANSFER_THRESHOLD`
- `ALT_SENDER_COUNT`

## Запуск

1. Установите Python 3.11+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env`.
4. Вставьте `BOT_TOKEN` от BotFather.
5. Укажите `ADMIN_CHAT_ID`.
6. Запустите:

```bash
python bot.py
```

Это игровая экономика, не настоящий банковский сервис.
